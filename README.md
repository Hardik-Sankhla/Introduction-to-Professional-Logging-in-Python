# Introduction to Professional Logging in Python

A comprehensive guide to professional logging practices in Python.

## Documentation

The full documentation is available at: https://hardik-sankhla.github.io/Introduction-to-Professional-Logging-in-Python/

## Local Development

### Prerequisites

- Python 3.12+
- pip

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Hardik-Sankhla/Introduction-to-Professional-Logging-in-Python.git
   cd Introduction-to-Professional-Logging-in-Python
   ```

2. Install dependencies:
   ```bash
   pip install mkdocs mkdocs-material
   ```

3. Serve the documentation locally:
   ```bash
   mkdocs serve
   ```

4. Open your browser to `http://localhost:8000`

### Building

To build the static site:

```bash
mkdocs build
```

The built site will be in the `site/` directory.
